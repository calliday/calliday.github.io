# calliday.github.io

My project portfolio - A simple, elegant landing page showcasing my projects.

## Structure

- `index.html` - Main landing page with project tiles
- `styles.css` - Stylesheet for the landing page
- `maven-cinemas/` - Maven Cinemas project directory (placeholder for React app)

## Projects

- **Maven Cinemas** - Cinema booking and management system (coming soon)

## Adding New Projects

To add a new project:
1. Create a new directory for the project (e.g., `my-project/`)
2. Add your project files, including an `index.html` entry point
3. Add a new project tile in `index.html` linking to `/my-project`

## GitHub Pages

This site is automatically deployed to GitHub Pages at: https://calliday.github.io
